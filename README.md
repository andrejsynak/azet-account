# Azet Account

Projekt zabezpečuje kompletný lifecycle Azet konta — od registrácie, cez prihlásenie, až po administráciu nastavení používateľa.

## Obsah projektu

- **Prihlásenie** — nový autentifikačný flow pre služby Azet.sk a online magazíny
- **Registrácia** — flow vytvorenia nového Azet konta
- **Administrácia konta** — správa údajov a nastavení Azet konta

## Kontext a cieľ

Tento projekt oddeľuje Azet konto od Pokec.sk konta, ktoré sa doteraz vytváralo súčasne. Dlhodobým cieľom je úplné oddelenie sveta Azet.sk a Pokec.sk — vrátane samostatnej administrácie aj databázy používateľov.

Azet bude fungovať ako **OAuth provider** pre nový Pokec.sk.

## Rozsah

| Oblasť | Popis |
|---|---|
| Prihlásenie | Azet.sk a online magazíny |
| Registrácia | Nové Azet konto |
| Administrácia | Správa Azet konta |
| Nastavenia | Používateľské nastavenia Azet konta |
| Separácia | Oddelenie od Pokec.sk konta a databázy |
